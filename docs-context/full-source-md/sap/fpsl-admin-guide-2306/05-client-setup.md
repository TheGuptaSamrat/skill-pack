# Client Setup

The guide recommends creating a separate FPSL client and highlights that BW is cross-client while the FPSL client concept is fully enabled.

The setup sequence includes:

1. create the client and assign the logical system
2. copy from client 000
3. create a background user and assign authorizations
4. create an RFC destination

Use this file for deeper operational sequencing, not as direct proof of local system values.
