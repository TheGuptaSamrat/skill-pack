# AMDP Supported Capabilities

- AMDP is the supported SAP mechanism for database procedures managed from ABAP and ADT.
- Profiling and result preview are valid SAP-supported techniques for troubleshooting and understanding AMDP runtime behavior.
- AMDP guidance in this repo should stay within supported ADT and HANA pushdown boundaries.
- Use official AMDP capabilities to justify pushdown technique choices, not to infer actual landscape artifacts.
