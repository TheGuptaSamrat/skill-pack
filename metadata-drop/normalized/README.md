# Normalized Metadata

This is the active metadata area used by skills by default.

Normalized artifacts placed here become the active working context for the skills.

Rules:

- stable filenames are preferred
- separate evidence from interpretation
- update `../manifest.csv` when replacing active normalized files
- if structure changes materially, update `../change-log.md`

Suggested contents:

- normalized CSV extracts
- compact metadata inventories
- reviewed mapping notes
- curated validation references
