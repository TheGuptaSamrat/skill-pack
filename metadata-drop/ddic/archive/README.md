# DDIC Archive

Use this folder for older DDIC exports that are no longer the active baseline.

Suggested pattern:

- `archive/<refresh-id>/tables.csv`
- `archive/<refresh-id>/fields.csv`
- `archive/<refresh-id>/keys.csv`
- `archive/<refresh-id>/data-elements.csv`
- `archive/<refresh-id>/domains.csv`
- `archive/<refresh-id>/manifest.csv`

Keep the active approved DDIC baseline in `../current/`.
