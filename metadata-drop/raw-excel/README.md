# Raw Excel Intake

Use this folder for trusted source metadata workbooks.

Rules:

- normalize them before using them as the default skill input
- use `manifest.csv` to track which normalized set comes from which workbook
- record the upload in `../manifest.csv`

Typical contents:

- FPSL metadata workbooks
- FSDM 2023 model workbooks
- CDS inventories exported as Excel
- configuration navigation or setup trackers
