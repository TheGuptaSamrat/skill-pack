# Metadata Sources

Useful evidence sources:

- DDIC exports for tables, fields, keys, domains, and data elements
- CDS DDL source and table function definitions
- FPSL core structures such as `/BA1/HFSPD`, `/BA1/HKTVR`, `/BA1/HKTVL`, `/BA1/F1_CON_FLAT`
- FSDM 2023 model structures and extension artifacts
- existing classes, interfaces, and AMDP methods

When metadata is incomplete:

- keep unresolved names as placeholders
- list what evidence is still required
